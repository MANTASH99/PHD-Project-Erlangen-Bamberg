## Project Structure

- **/src**: Contains all main Python source code for data processing, model training, and explanation generation.
- **/data**: Intended for storing datasets. Note: Large data files are not tracked by Git. Please see the setup instructions to download the required corpora.
- **/notebooks**: Contains Jupyter notebooks for exploratory data analysis (EDA), results visualization, and qualitative analysis.
- **/models**: Intended for storing trained model checkpoints. Note: Model files are not tracked by Git.
